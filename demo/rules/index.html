CSS Compressor Rules

  • Atrule URL Trim

    Author: Corey Hart
    Converts at-rule one liner url's to strings.
    Before: 
    @import url(other.css)
    After: 
    @import "other.css"

  • Border Combinations

    Author: Corey Hart
    Combines border directionals into single border property.
    Before: 
    .example {
	border-top: 1px solid red;
	border-left: 1px solid red;
	border-right: 1px solid red;
	border-bottom: 1px solid red;
}
    After: 
    .example {
	border: 1px solid red;
}

  • Border Radius Combinations

    Author: Corey Hart
    Combines border radius directionals into their shorter alternative.
    Before: 
    .example {
	border-top-left-radius: 10px;
	border-top-right-radius: 10px;
	border-bottom-left-radius: 10px;
	border-bottom-right-radius: 10px;
}
    After: 
    .example {
	border-radius: 10px;
}

  • Border Radius Shorthand

    Author: Corey Hart
    Converts shorthand border radius properties.
    Before: 
    .example {
	border-radius: 10px 10px 10px 10px;
}
    After: 
    .example {
	border-radius: 10px;
}

  • Border, Outline Style Combinations

    Author: Corey Hart
    Combine border & outline expanded properties into shorthand.
    Before: 
    .example {
	border-top-width: 2px;
	border-top-style: dashed;
	border-top-color: blue;
}
    After: 
    .example {
	border-top: 2px dashed blue;
}

  • Class Attribute to Selector

    Author: Corey Hart
    Converts class attributes to class selectors.
    Before: 
    [class='example'] {
	color: red;
}
    After: 
    .example {
	color: red;
}

  • Color to Hex

    Author: Corey Hart
    Converts color names to shorter hex representation.
    Before: 
    .example {
	color: aliceblue;
}
    After: 
    .example {
	color: #f0f8ff;
}

  • Comma Repeats

    Author: Corey Hart
    Removes repeated selectors separated by commas.
    Before: 
    html body, html body {
	color: red;
}
    After: 
    html body {
	color: red;
}

  • Common Rules

    Author: Corey Hart
    Combines rule sets with matching rules.
    Before: 
    .example {
	color: red;
	font-size: 2px;
}

.match {
	color: red;
	font-size: 2px;
}
    After: 
    .example, .match {
	color: red;
	font-size: 2px;
}

  • Common Selectors

    Author: Corey Hart
    Combines rule sets with matching selectors.
    Before: 
    .example {
	border: 1px dashed green;
}

.example {
	cursor: pointer;
}
    After: 
    .example {
	border: 1px dashed green;
	cursor: pointer;
}

  • Duplicate Properties

    Author: Corey Hart
    Removes multiply defined properties.
    Before: 
    .example {
	color: red;
	margin: 10px;
	color: green;
}
    After: 
    .example {
	margin: 10px;
	color: green;
}

  • Empty Values

    Author: Corey Hart
    Removes properties that don't have values.
    Before: 
    .example {
	color: red;
	font-size:;
}
    After: 
    .example {
	color: red;
}

  • Font Combinations

    Author: Corey Hart
    Combines font declarations into single property.
    Before: 
    .example {
	font-weight: bold;
	font-size: 12px;
	font-family: Helvetica;
}
    After: 
    .example {
	font: bold 12px Helvetica;
}

  • Font Family Quotes

    Author: Corey Hart
    Removes quotes around font family names.
    Before: 
    .example {
	font-family: "Arial Black";
}
    After: 
    .example {
	font-family: Arial Black;
}

  • Font Weight Conversion

    Author: Corey Hart
    Converts font weight strings to their numeric counterparts.
    Before: 
    .example {
	font-weight: normal;
}
    After: 
    .example {
	font-weight: 400;
}

  • Gradient Compression

    Author: Corey Hart
    Breaks down gradient properties and runs compressions on individual parts.
    Before: 
    .example {
	background: linear-gradient( top, black 0%, #dddddd 100% );
}
    After: 
    .example {
	background: linear-gradient(top,#000 0%,#ddd 100%);
}

  • Hex to Color

    Author: Corey Hart
    Converts hex codes to short color names.
    Before: 
    .example {
	color: #fffff0;
}
    After: 
    .example {
	color: ivory;
}

  • Hex to Safe Color

    Author: Corey Hart
    Converts hex codes to short color names that are cross browser safe.
    Before: 
    .example {
	color: #000080;
}
    After: 
    .example {
	color: navy;
}

  • ID Attribute to Selector

    Author: Corey Hart
    Converts id attributes to id selectors.
    Before: 
    [id='example'] {
	color: red;
}
    After: 
    #example {
	color: red;
}

  • Leading Zeroes

    Author: Corey Hart
    Removes unecessary leading zeroes from values.
    Before: 
    .example {
	font-size: 0.2px;
}
    After: 
    .example {
	font-size: .2px;
}

  • List Style Combinations

    Author: Corey Hart
    Combines list-style properties into their shorter alternative.
    Before: 
    .example {
	list-style-type: square;
	list-style-position: outside;
	list-style-image: none;
}
    After: 
    .example {
	list-style: square outside none;
}

  • Lowercase Hex

    Author: Corey Hart
    Lowercases hex color values for better gzip compression.
    Before: 
    .example {
	color: #ABCDEF;
}
    After: 
    .example {
	color: #abcdef;
}

  • Lowercase Properties

    Author: Corey Hart
    Lowercases property names for better chance at compression during gzip.
    Before: 
    .example {
	COLOR: red;
}
    After: 
    .example {
	color: red;
}

  • Lowercase Selectors

    Author: Corey Hart
    Converts element selectors to lowercase.
    Before: 
    BODY {
	color: red;
}
    After: 
    body {
	color: red;
}

  • Margin, Padding Combinations

    Author: Corey Hart
    Combines margin and padding directionals into their shorter alternative.
    Before: 
    .example {
	margin-top: 5px;
	margin-right: 5px;
	margin-bottom: 5px;
	margin-left: 5px;
}
    After: 
    .example {
	margin: 5px;
}

  • Margin, Padding Shorthand

    Author: Corey Hart
    Combines Margin/Padding shorthand directionals.
    Before: 
    .example {
	margin: 3px 3px 3px 3px;
}
    After: 
    .example {
	margin: 3px;
}

  • None Conversions

    Author: Corey Hart
    Converts none values to zeroes for allowed properties.
    Before: 
    .example {
	border: none;
}
    After: 
    .example {
	border: 0;
}

  • Order Atrules

    Author: Corey Hart
    Moves charset and import declarations to the top of the stylesheet (to help with concatenation).
    Before: 
    .example {
	color: red;
}

@charset 'utf-8'
@import 'home.css'
    After: 
    @charset 'utf-8'
@import 'home.css'

.example {
	color: red;
}

  • Rect Shape

    Author: Corey Hart
    Compressing rect shape declarations
    Before: 
    rect( 1.0px, 2.0px, 3.0px, 4.0px )
    After: 
    rect(1px,2px,3px,4px)

  • RGB to Hex

    Author: Corey Hart
    Converts RGB to it's Hex code alternative.
    Before: 
    .example {
	color: rgb(214,123,214);
}
    After: 
    .example {
	color: #d67bd6;
}

  • Shrink Hex

    Author: Corey Hart
    Converts long hex codes to short hex codes.
    Before: 
    .example {
	color: #000000;
}
    After: 
    .example {
	color: #000;
}

  • Sort Multi Selectors

    Author: Corey Hart
    Sorts comma separated selectors for a better chance at compression during gzip.
    Before: 
    span, b, div, a {
	color: red;
}
    After: 
    a, b, div, span {
	color: red;
}

  • Sort Properties

    Author: Corey Hart
    Sorts property-values for a better chance at compression during gzip.
    Before: 
    .example {
	padding: 10px;
	background: green;
	margin: 10px;
	color: red;
}
    After: 
    .example {
	background: green;
	color: red;
	margin: 10px;
	padding: 10px;
}

  • Strict ID

    Author: Corey Hart
    Removes entire selector before last ID selector.
    Before: 
    html body #myid div {
	color: red;
}
    After: 
    #myid div {
	color: red;
}

  • Strip Comments

    Author: Corey Hart
    Removes all comments from style sheet, with the exception of comments that are prefixed with a bang '/*!'
    Before: 
    /*
	Multi Line Comment
*/
.example {
	color: red;
}
    After: 
    .example {
	color: red;
}

  • Strip Empty Branches

    Author: Corey Hart
    Removes all branches with no rules.
    Before: 
    .example {
	color: red;
}

.empty {}

body {
	font: Helvetica;
}
    After: 
    .example {
	color: #f0f8ff;
}

body {
	font: Helvetica;
}

  • Trailing Zeroes

    Author: Corey Hart
    Removes unecessary trailing zeroes from values.
    Before: 
    .example {
	font-size: 5.0px;
}
    After: 
    .example {
	font-size: 5px;
}

  • Trim Selector Attribute Quotes

    Author: Corey Hart
    Remove attribute quotes when they are not needed.
    Before: 
    .example[attr='nospace'] {
	color: red;
}
    After: 
    .example[attr=nospace] {
	color: red;
}

  • Unit Suffix

    Author: Corey Hart
    Removes unecessary suffix from zero unit values.
    Before: 
    .example {
	font-size: 0px;
}
    After: 
    .example {
	font-size: 0;
}

  • URL Trim

    Author: Corey Hart
    Removes quotes around url wrappers.
    Before: 
    .example {
	background-image: url("http://www.google.com/image.png");
}
    After: 
    .example {
	background-image: url(http://www.google.com/image.png);
}
